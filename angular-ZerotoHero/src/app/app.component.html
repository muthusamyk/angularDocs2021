Components

  • Components are the basic building block of an angular
  • It's where you write your bind code
  • Binds the View and model

Data Binding

  • Communication b/w typescript Component and the html template
  • types of data binding :
    • String Interpolation --> Send data from TypeScript component to Html template
    • Property Binding [] --> we can directly bind the values to the property ex: [src] for image tag
    • Event Binding () --> pass the data from html template to typescript ex: input value pass to typescript from html... Also used for mouse movement keyup clicks and etc
    • Two way Binding [()] --> By using ngModel to bind data both direction

Directives

  • It helps to attach behaviour to your html DOM.
    • Structural Directives --> Modify the DOM
    • Attribute Directives --> Modify the Properties of the DOM
